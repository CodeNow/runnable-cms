# The CMS for Runnable.com

### Requirements

- Ruby (v2.2 or above)
- RubyGems
- Bundler (bundler.io)

### Development

1. Clone this repo.
2. Run `rake serve`
3. Open `localhost:4000/use-cases` in your browser.

## Deploying

Any changes to master will automatically deploy to GitHub Pages (AKA Prod @ runnable.com/use-cases/)
